웹페이지(웹문서)를 제작하는 언어
1. HTML
    - 웹문서를 작성하는 언어
    - 페이지의 골격을 작성
    - 페이지의 출력내용을 작성
2. CSS
    - 디자인 언어
    - HTML페이지 + 서식을 작성하는 역할
3. JavaScript
    - 프로그래밍 언어(Java같은)
    - 웹페이지 + 프로그래밍 기능
    
HTML(Hyper Text Markup Language)
- 하이퍼 텍스트 마크업 언어
- 프로그래밍 언어(X) →  프레젠테이션  언어(O) : 무언가를 표현하는 기능을 가진 언어
- Hyper Text → 링크
- Markup → 언어 표기법 : <키워드>
    - HTML, XHTML, XML, VoiceXML, **ML
- SGML → 메타언어 : 다른 언어를 정의할 수 있는 기능을 가진 언어
    - HTML, XML
